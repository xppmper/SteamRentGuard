SteamRentGuard v0.53 BETA 🛡️

SteamRentGuard — это специализированное защитное ПО, разработанное для обеспечения безопасности при аренде Steam-аккаунтов. Программа осуществляет активный мониторинг системы, предотвращая использование читов и доступ к ресурсам их распространения.

Данное ПО необходимо, если вы хотите сдать свой аккаунт Counter-Strike 2 в аренду на популярных маркетплейсах. Вы просто скидываете покупателю `SteamRentGuard.exe`, настраиваете уведомления, и с шансом 80% на вашем аккаунте не будет читеров.

---

## ⚙️ Настройка перед использованием (Что редактировать)

Чтобы программа присылала уведомления именно вам, нужно внести изменения в исходный код перед компиляцией:

### 1. В файле `SteamRentGuard.cpp`:
* **Строка №17:** Замените текст в кавычках на токен вашего Telegram-бота. Токен бота можно найти у бота @BotFather
* **Строка №18:** Замените число на ваш личный `chatId` (ID вашего чата с ботом).  Айди в телеграме можно найти в боте @userinfobot

## 🚀 Инструкция по запуску

1. **Компиляция**: Откройте проект в Visual Studio и скомпилируйте его (режим **Release x64**).
2. **Передача**: Передайте полученный файл `SteamRentGuard.exe` арендатору.
3. **Запуск**: Арендатор ОБЯЗАН запустить программу **от имени администратора**.
4. **Работа**: После запуска программа скроется из панели задач. Если арендатор попытается запустить чит или зайти на сайт софта — программа закроет Steam и пришлет вам уведомление.

## 🔥 Основные возможности:
* 🚀 **Активный мониторинг процессов**: База данных включает более 90 паттернов известных читов (Midnight, Xone, Neverlose, Interium и др.).
* 🌐 **Блокировка веб-ресурсов**: Автоматическая модификация файла `hosts` для запрета доступа к сайтам читов.
* 🔎 **Защита через заголовки окон**: Мониторинг вкладок браузера. При попытке зайти в разделы "Buy", "Support" или "Account" на сайтах читов, система мгновенно реагирует.
* 📲 **Telegram-уведомления**: Отчеты об обнаруженных угрозах приходят мгновенно.
* 🛡️ **Экстренное завершение**: При угрозе Steam и CS2 закрываются принудительно.

## 🛠 Технические особенности:
- Язык: **C++** (WinAPI).
- Автоматическая очистка **DNS-кэша** для активации сетевых блокировок.
- Скрытый режим работы.
- Проверка прав администратора при старте.

## 📋 Требования:
- ОС: Windows 10/11.
- Запуск от имени администратора.
